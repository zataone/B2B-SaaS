#B2B SaaS Dashboard (Multi-Tenant Workspace)

Aplikasi manajemen proyek atau keuangan skala perusahaan (seperti Notion atau Slack versi mini).

Fitur Utama: Sistem otentikasi multi-peran (Admin, Manajer, Staf), manajemen data dengan grafik interaktif (charts), fitur ekspor laporan ke PDF/Excel, dan pengaturan organisasi/tim.

Keunggulan Sistem: Keamanan ketat berkat TypeScript, performa data yang cepat dengan SSR, dan optimasi komponen besar menggunakan ekosistem React yang matang.
